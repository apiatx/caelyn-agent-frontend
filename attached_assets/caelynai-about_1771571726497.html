What CaelynAI Is

A decision engine for active traders

CaelynAI replaces the 20-tab workflow. It pulls real-time price action, technical indicators, fundamental data, derivatives positioning, social sentiment, and macro signals — then a scoring engine ranks everything by conviction before an AI strategist synthesizes the output into specific trades with entries, stops, and targets.

Every recommendation passes a simple test: would a portfolio manager with $2M of their own capital actually size into this position? If the answer is no, it doesn't make the cut. If nothing qualifies, CaelynAI says so. Capital preservation is a valid output.

Architecture

How the pipeline works

Every scan follows the same disciplined flow. No shortcuts, no vibes-based analysis.

Routing — Preset intents resolve deterministically. Free-text queries go through an orchestrator that builds a data-gathering plan based on asset classes, modules, and filters.
Data Gathering — Parallel API calls with per-source timeouts, circuit breakers on repeated failures, tiered fallbacks, and a caching layer. If a source is down, the pipeline degrades gracefully — never returns blank.
Regime Detection — Macro regime (risk-on / risk-off / neutral) is computed from VIX, DXY, yield curve, Fear & Greed. This regime gates which asset classes and sectors get elevated or penalized.
Deterministic Scoring — Each candidate is scored 0–100 using weighted components: 30% Technical (Weinstein stage, trend, volume, pattern), 30% Catalyst (specificity, time-bound, repricing potential), 20% Sector Alignment, 10% Social Momentum, 10% Risk/Reward.
AI Synthesis — Claude receives the scored, compressed data and produces the final output: trade plans with entry, stop, target, R:R, conviction, position sizing tier, and a thesis grounded in the data — not generated from memory.

Crypto Intelligence

Built for derivatives traders

The crypto module treats BTC as the only true investment. Everything else is a trade based on hype cycles paired with real technical momentum or catalysts. The system is built around this philosophy.

HyperLiquid Perps
Real-time funding rates, open interest, and volume across every perpetual future on HyperLiquid. Squeeze candidates (extreme negative funding + rising price), crowded longs (liquidation risk), and funding divergences — the highest-signal trades in crypto.
X Sentiment via Grok
Social velocity detection — not just who's mentioned, but whose mentions are accelerating fastest. A micro-cap going from 100 to 1,000 mentions is higher signal than a large-cap with steady 5,000. Grok scans X in real-time and returns structured sentiment with BTC context.
altFINS TA
Pre-computed technical analysis across 2,000+ crypto assets. RSI, MACD, SMA/EMA crossovers, chart patterns, candlestick patterns, support/resistance, and trend scores. CaelynAI never calculates indicators from raw candles — it uses institutional-grade pre-computed signals.
Narrative Rotation
Hot category tracking from CoinGecko and CMC. Which sectors (AI, gaming, RWA, meme coins, L2s, DePIN) are gaining market cap? Dual-trending across both platforms is the strongest momentum signal.
